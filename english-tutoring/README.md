## 使用顺序（零基础唯一正确顺序）

**音标 → 词汇 → 基础语法 → 阅读 → 练习巩固**

1. 先用 4 周吃透音标（01），做到见词能读；
2. 同步每天背 10 个分类词汇（03）；
3. 按语法框架（02）逐章学习，每学完一章：
   - 精读对应的阅读文章（04）；
   - 完成对应练习题（06）并订正；
4. 每周 2 次情景对话操练（05）；
5. 按学习计划（07）控制节奏，滚动复习。
