# 第七部分：40周补习计划与一对一教学法（零基础提分路线图）

## 一、总体路线（三个阶段）

| 阶段 | 周次 | 目标 | 核心内容 |
|------|------|------|------|
| 打地基 | 第1-8周 | 见词能读、掌握300词、能写简单句 | 音标 + 词汇 + be动词/一般现在时 |
| 建框架 | 第9-24周 | 掌握初一初二全部语法、词汇量800+ | 八大时态前五个 + 代词/介词/比较级 + 阅读入门 |
| 冲提高 | 第25-40周 | 对接中考题型、词汇量1200+ | 完成时/被动/从句 + 阅读写作专项 |

**每周课时建议**：一对一辅导 2 次 × 1.5 小时 + 学生自主打卡每天 30 分钟（背词15分钟 + 朗读15分钟）。

## 三、一对一教学法（给家长/老师）

### 1. 每节课标准流程（90分钟）
1. **复习检测（15分钟）**：听写上次课单词10个 + 口头提问上次语法点3题；
2. **新课讲解（30分钟）**：讲一个语法点，规则不超过3条，每条立即配2个例句让学生仿造；
3. **阅读应用（20分钟）**：精读配套文章，让学生在文中「找语法」（如：圈出所有三单动词）；
4. **口语操练（15分钟）**：情景对话跟读+替换练习；
5. **总结布置（10分钟）**：学生用自己的话复述本课规则，布置打卡任务。

### 2. 零基础学生五大教学原则
1. **音标先行，绝不跳过**：单词记不住的根源是不会拼读。前4周只抓音标，不着急学语法；
2. **一次只讲一个点**：零基础学生每节课只能吸收一个语法点，宁慢勿快；
3. **例句代替术语**：不说"第三人称单数需要动词屈折变化"，而说"he/she/it 后面动词加 s，跟我读三个句子"；
4. **高频重复+滚动复习**：单词按 1-2-4-7-15 天间隔滚动听写；语法每学新章必带练旧章；
5. **正反馈驱动**：每次测验只和学生自己上次的成绩比，进步就表扬，建立"我能学好英语"的信心。

### 3. 常见问题应对
| 问题 | 应对方法 |
|------|------|
| 单词背了就忘 | 检查是否会音标拼读；改为"音标拼读+例句造句"记忆法，禁止死记字母 |
| 三单老是忘加s | 每天口头快问快答10句（I play → He ___?），形成条件反射 |
| 句子语序混乱（中式英语） | 用五大基本句型模板"填空造句"：先给框架 主+谓+宾，再让学生填词 |
| 阅读看不懂 | 退回上一级难度文章；先积累词汇量到500再推进 |
| 不敢开口 | 从跟读模仿开始，录音回放让学生自己听进步 |

### 4. 每月验收指标
- 词汇：每月新增背诵 120-150 词，听写正确率 ≥ 80%；
- 语法：当月练习题正确率 ≥ 80%；
- 阅读：能独立读懂当月配套文章的 70% 以上内容；
- 口语：能脱稿完成当月 2 组情景对话。

## 四、推荐配套资源

1. **教材同步**：以学生在校教材（人教版/外研版初一）为主线，本套资料作系统补充，进度尽量与学校单元对应；
2. **听力磨耳朵**：每天10分钟，推荐教材配套录音、《新概念英语第一册》录音（语速慢、句型标准）；
3. **词典工具**：初期用有道词典/欧路词典查音标和例句，要求学生先自己拼读再听发音核对；
4. **分级读物**：牛津《书虫》入门级、《新概念英语第一册》课文（每篇都是极佳的背诵材料）；
5. **App 打卡**：百词斩/不背单词（选"初中人教版"词库），每天15分钟。
